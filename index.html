Projects

Graph Res

A resistance calculator built with jQuery.

I love working with electronics but one point of frustration is memorizing the resistor band codes. I wanted to build a resistance calculator that was clean and simple. To determine the value of a resistor simply select the correct number of bands and enter the appropriate band colors.

Built Primarily With: HTML, CSS, jQuery

Partner Dashboard

My final project for the Code Fellows Bootcamp.

This application allows business partners of the Code Fellows program to connect with current students and the graduates of the program that are looking for a job. Utilizes invitation sign-up and filter based search for narrowing students down based on their skill set.

Built Primarily With: Ruby, Rails, Bootstrap, RSpec

Family United

My entry for the PayPal Battlehack 2013.

Stay in touch with your family with group text messaging, email, and calls all with one phone number. Uses the Twilio API for messaging and the Paypal API for payment processing. This is a work in progress!

Built Primarily With: Ruby, Rails, Bootstrap, CoffeeScript, jQuery

Questions From A Ghost

A social experiment built with Ruby on Rails.

My first Rails project! I wanted to see how people reacted to random notes left in public places. A note has a question, a URL to my site, and a passcode. People who find notes could answer the question online and the site will populate with the answer.
